select 태그와 video 태그
*select 태그의 option 태그는 선택이 가능할까? option 태그는 선택이 불가능 하지만 value 속성은 선택가능
*video 태그의 source 태그는 선택이 불가능함!, src,type 만 선택이 가능함
*srcipt 에서 선택하는 방법 : select.value / video.src

video 활용하기

  1. video 태그는 "인라인요소" input,img,select처럼 스스로 너비와 높이를 가진다
  2. video 태그에 width="640px" height="480px" , validator확인시 에러 발생
  3. style 적용이 방식이 아니라 사용자가 직접 너비와 높이를 지정하여 작업함
  4. 위와 같은 방식은 브라우저에서 이미 px로 인식하기 떄문에 오류가 발생함!
  5. * 제일 안전한 방식은 style 에서 수정하는 작업이 가장 안전하다!
  6. mp4,ogg,webM 이외의 확장자는 재생불가능

mp4 확장자가 재생되지 않는 이유

mp4는 모든 브라우저에서 지원이 가능함, 익스플로러는 10버전 부터 재생이 가능함!
: 모바일용과 pc용으로 구분했을 때 모바일용이 재생 불가능한 경우가 있음
: 모바일용으로 인코딩을 새로 해주어야 재생이 가능하다.
비디오가 재생되지 않을 때, 대체 텍스트와 마찬가지로, poster 속성을 사용하여 이미지로 대체함!
